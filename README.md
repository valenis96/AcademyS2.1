# AcademyS2.1

## Exercicis bàsics JavaScript

This project contains a collection of basic JavaScript exercises structured into **7 different script files**.

The UI is a single HTML page where each exercise can be executed by clicking the corresponding element.

<img width="1918" height="826" alt="image" src="https://github.com/user-attachments/assets/cb89edb5-4401-4d49-85e6-bf84bc121a62" />

### Objectives

- Understand and practice the use of arrow functions.

- Learn and apply the ternary operator in different contexts.

- Understand and work with callbacks.

- Learn and use the Rest and Spread operators.

- Become familiar with loops and array transformations.
